# Merge Checklist

- [ ] `MG-01` PR head/base revisions and scope match reviewed evidence.
- [ ] `MG-02` G0–G7 applicable gates pass; applicability exceptions include evidence.
- [ ] `MG-03` Frozen spec and implementation-plan items trace to the final diff and tests.
- [ ] `MG-04` No critical/high finding is open; medium dispositions satisfy policy.
- [ ] `MG-05` Required build, type, lint, test, migration, security, and performance checks pass.
- [ ] `MG-06` Contracts, consumers, manifests, locks, generated outputs, and graph are synchronized.
- [ ] `MG-07` Documentation, decisions, terminology, module ownership, and release notes are current.
- [ ] `MG-08` Migration, compatibility, rollout, feature flags, and rollback are validated where applicable.
- [ ] `MG-09` No secret, sensitive data, debug code, unrelated change, or unauthorized generated file is included.
- [ ] `MG-10` Required independent and human approvals exist for the exact candidate revision.
